Package MailWizzApi\Cache

Classes summary
MailWizzApi_Cache_Abstract MailWizzApi_Cache_Abstract is the base class that all the caching classes should extend.
MailWizzApi_Cache_Apc MailWizzApi_Cache_Apc makes use of the APC extension in order to cache data in memory.
MailWizzApi_Cache_Database MailWizzApi_Cache_Database makes use of the PHP's PDO extension in order to cache data into a database.
MailWizzApi_Cache_Dummy MailWizzApi_Cache_Dummy is used for testing purposes, when you use the sdk with cache but don't want to really cache anything.
MailWizzApi_Cache_File MailWizzApi_Cache_File makes use of the file system in order to cache data.
MailWizzApi_Cache_Xcache MailWizzApi_Cache_Xcache makes use of the xcache extension in order to cache data in memory.
